Preciso testar um CRUD?
==========================

- O que é CRUD?
  - Create, Read, Update, Delete
  - normalmente é um código simples

- Pergunta errada!
  - simples ou complexo não faz diferença
  - por que você quer testar?
  - lógica de fluxo e lógica de estado

- Custo x benefício
  - é um código que vai sofrer muita alteração?
  - código estável x código instável
  - priorize o ROI
